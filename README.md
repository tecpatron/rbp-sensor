# RBP Sensor

Ultima actualización: 2025-12-04

## Objetivo
_El proyecto surge como respuesta a la necesidad de contar con un dispositivo accesible y fácil de usar que permita monitorear en tiempo real las condiciones ambientales en interiores, especialmente temperatura y humedad. Al disponer de esta información, los usuarios podrán tomar decisiones informadas, como mejorar la ventilación, evitar niveles excesivos de humedad o ajustar sistemas de climatización para obtener un ambiente más confortable y saludable._

## Software
* Arduino UNO
* PyCharm
